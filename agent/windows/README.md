# Автозапуск агента на киоск-ПК (Windows)

Запускает агент в **kiosk-host режиме**: агент сам открывает Chrome с интерфейсом
киоска, печатает локально, всё на одном `http://localhost:8089` (без mixed-content).

> Почему не служба Windows? Служба работает в session 0 без рабочего стола —
> Chrome не появится на экране. Поэтому автозапуск в **пользовательской сессии**
> через папку автозагрузки.

## Установка (один раз)

1. Скопируй на киоск в папку `C:\agent\`:
   - `ndpi-agent.exe` (свежий, `ndpi-agent.exe -version` → должно быть `0.2.0`)
   - `start-agent.bat`
   - `run-hidden.vbs`

2. В `start-agent.bat` впиши **точное имя принтера** в `AGENT_PRINTER_NAME`
   (PowerShell: `Get-Printer | Select Name`).

3. Разблокируй exe, если Defender пометил: ПКМ по `ndpi-agent.exe` → Свойства →
   галочка «Разблокировать» (Unblock). При необходимости добавь `C:\agent` в
   исключения Защитника Windows.

4. Автозапуск: `Win+R` → `shell:startup` → в открывшуюся папку положи **ярлык**
   на `C:\agent\run-hidden.vbs` (ПКМ по vbs → Отправить → Рабочий стол (ярлык),
   потом перетащи ярлык в папку автозагрузки).

5. **Автологин** (чтобы автозагрузка сработала без ввода пароля): `Win+R` →
   `netplwiz` → снять «Требовать ввод имени пользователя и пароля» → ввести
   пароль киоск-учётки. (На kiosk-ПК обычно уже включён.)

6. Перезагрузка → Chrome сам откроется в режиме киоска, агент работает в фоне
   (без чёрного окна), лог пишется в `C:\agent\agent.log`.

## Проверка / отладка
- Логи: `C:\agent\agent.log` (там `agent started ...`, ошибки печати).
- Список принтеров: открыть `http://localhost:8089/printers` в браузере.
- Здоровье: `http://localhost:8089/health`.
- Выбор принтера: на киоск-странице 5 быстрых тапов по логотипу «NP» → настройки.

## Надёжнее (перезапуск при падении) — Планировщик заданий
Вместо папки автозагрузки можно Task Scheduler:
- Триггер: «При входе в систему» (текущий пользователь).
- Действие: запустить `wscript.exe "C:\agent\run-hidden.vbs"`.
- Параметры: «Перезапускать при сбое» каждые 1 мин, до 3 раз.
- «Выполнять только для вошедшего пользователя» (нужен рабочий стол для Chrome).
