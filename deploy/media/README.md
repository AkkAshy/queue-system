# Board media (табло)

Сюда кладутся видео для левой медиа-зоны табло — **локально, без интернета**.
Можно **одно видео или несколько** — табло крутит их по кругу (плейлист).

## Как использовать
1. Положи сюда один или несколько видеофайлов: `.mp4`, `.webm`, `.mov`, `.m4v`.
   Формат: H.264/AAC, желательно 720p. Звук не нужен — табло играет без звука.
2. Перезапусти nginx — файлы подхватятся сразу, пересборка табло НЕ нужна:
   ```cmd
   docker compose -f deploy/docker-compose.local.yml --env-file deploy/.env.local up -d nginx
   ```
3. Открой табло — ролики играют один за другим, по кругу.

## Порядок роликов
Сортировка по имени файла. Чтоб задать порядок — нумеруй:
```
01-intro.mp4
02-priyom.mp4
03-obyavlenie.mp4
```

## Как это работает
- nginx раздаёт папку по пути `/media/` и отдаёт JSON-список файлов
  (autoindex) — см. `deploy/nginx/local.conf`.
- Табло читает список, фильтрует видео, строит плейлист и играет по кругу.
  Папка `/media/` задаётся build-аргом `NEXT_PUBLIC_BOARD_MEDIA_DIR`.
- Если папка пустая/недоступна → табло **автоматически** откатывается на
  YouTube-URL из админки, а если и его нет — на брендовый плейсхолдер.

## Добавить / убрать ролик
Просто кинь или удали файл и перезапусти nginx. Всё.

> Сами видеофайлы в git НЕ коммитятся (см. `.gitignore`) — только этот README.
