# Board media (табло)

Сюда кладётся видео, которое крутится в левой медиа-зоне табло — **локально, без интернета**.

## Как использовать
1. Положи файл сюда с именем **`board.mp4`** (H.264/AAC, желательно 720p, без звука или muted — табло всё равно играет без звука).
2. Перезапусти nginx (или весь стек): файл подхватится сразу, пересборка табло не нужна — nginx раздаёт его как `/media/board.mp4`.
   ```cmd
   docker compose -f deploy/docker-compose.local.yml --env-file deploy/.env.local up -d nginx
   ```
3. Открой табло — играет локальное видео.

## Как это работает
- nginx раздаёт эту папку по пути `/media/` (см. `deploy/nginx/local.conf`).
- Табло берёт путь из build-арга `NEXT_PUBLIC_BOARD_VIDEO=/media/board.mp4` (см. `docker-compose.local.yml`).
- Если файла нет (404) → табло **автоматически** откатывается на YouTube-URL из админки, а если и его нет — на брендовый плейсхолдер.

## Сменить ролик
Просто замени `board.mp4` новым файлом с тем же именем и перезапусти nginx. Всё.

> Файлы видео в git НЕ коммитятся (см. `.gitignore`) — только этот README.
