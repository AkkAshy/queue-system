# План доработки под ТЗ (NDPI Navbat) — Local-first

Архитектура: **локальный бокс на объекте работает офлайн по LAN, облако = админка + статистика, синк без конфликтов.**
Каждый шаг: **действие** → **критерий готово**. Отмечай `[x]` и иди дальше.

---

## Архитектура (зафиксировано)

```
        ОБЛАКО (nmpi.avtoxizmet.uz)                  ОБЪЕКТ (LAN)
  ┌──────────────────────────────┐          ┌──────────────────────────────┐
  │ Админка (откуда угодно, auth) │          │ Локальный бокс (NUC)          │
  │  • правит КАТАЛОГ + настройки │          │  Django+PG+Redis+nginx        │
  │  • аккаунты                   │          │  раздаёт киоск/табло/пульт    │
  │  • СТАТИСТИКА / отчёты / Excel │          │  по LAN, работает ОФЛАЙН      │
  │  • аудит                      │          │                               │
  └──────────────┬───────────────┘          └───────────────┬──────────────┘
                 │  catalog (cloud→local, pull)              │
                 │ ─────────────────────────────────────────►│  упорт в локальную БД
                 │                                           │
                 │  events (local→cloud, push)               │
                 │ ◄─────────────────────────────────────────│  талоны/сессии/аудит
  агрегация ◄────┘     (идемпотентно по UUID)                └─ киоск-ПК: агент печати (USB)
```

**Кто что пишет (нет конфликтов):**
- **Каталог** (Hall, Category, Service, Counter, User, Settings, Schedule) — пишет ТОЛЬКО облако → локальный читает.
- **События** (Ticket, OperatorSession, AuditLog, DailyCounter) — пишет ТОЛЬКО локальный → облако принимает и агрегирует.

**Офлайн:** локальный работает на закэшированном каталоге; обновления каталога подтягиваются, когда есть инет. Выдача/вызовы не страдают.
**Realtime удалённо** — отдельная будущая фаза (сейчас realtime только по LAN).

## Фаза A — Локальный деплой (фундамент)
Поднять текущий стек на локальном боксе, устройства по LAN. Без этого синк негде крутить.

- [ ] `deploy/docker-compose.local.yml` — стек без cloud-прокси: db+redis+backend+4 фронта+nginx, биндинг на LAN-интерфейс.
      **Готово:** на NUC `docker compose -f deploy/docker-compose.local.yml up -d` поднимает всё.
- [ ] Фронты собираются с `NEXT_PUBLIC_API` = адрес локального бокса (или relative через локальный nginx).
      **Готово:** киоск/табло/пульт открываются с других устройств LAN по `http://<box-ip>/`.
- [ ] Печать: агент на киоск-ПК (уже есть), киоск-страница шлёт на `localhost:8089` (CORS уже добавлен).
      **Готово:** талон печатается с киоск-ПК при бэке на NUC.
- [ ] Удалённый доступ к NUC для обслуживания (Tailscale/SSH).
      **Готово:** можно зайти на NUC удалённо, `docker compose pull && up`.

**Готово, когда:** вся система работает на NUC по LAN без интернета.

## Фаза D — Синк-движок (ядро local-first) ❌ [новое]
- [ ] `Ticket.finished_at` (+ проставлять в `finish`) — для времени обслуживания. Миграция.
      **Готово:** завершение талона пишет `finished_at`.
- [ ] Флаг `synced` (+`synced_at`) на Ticket/OperatorSession/AuditLog.
      **Готово:** новые события создаются с `synced=false`.
- [ ] **Cloud→local (каталог):** облако отдаёт `GET /api/sync/catalog?since=<v>` (изменённые Hall/Category/Service/Counter/User/Settings/Schedule + версия). Локальный апсёртит по PK.
      **Готово:** правка услуги в облаке появляется на локальном после синка.
- [ ] **Local→cloud (события):** локальный батчем `POST /api/sync/events` (несинканые). Облако апсёртит по UUID (идемпотентно). На ack → `synced=true`.
      **Готово:** талоны/сессии/аудит с локального появляются в облаке без дублей; повтор не плодит.
- [ ] **Синк-воркер** на локальном (Celery-beat или loop): каждые N сек при наличии инета — push events, pull catalog. Бэкофф при офлайне.
      **Готово:** выдернул инет → копит; вернул → догнал, дубликатов нет.
- [ ] Конфиг режима: `SYNC_ROLE=local|cloud`, `CLOUD_URL`, sync-токен.
      **Готово:** один кодбейс, режим из env.

**Готово, когда:** локальный офлайн копит события и синкает в облако при связи; каталог из облака доезжает на локальный. Дублей и конфликтов нет.
