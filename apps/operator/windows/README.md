# Пульт оператора на ПК — Chrome app-mode (Win 7 и Win 10)

Пульт запускается как **окно-виджет в Chrome** (`--app`), без вкладок и адресной
строки. Работает и на Win 7, и на Win 10 — **WebView2/Tauri не нужен**.

> Почему не Tauri: Tauri v2 требует WebView2, а Microsoft прекратил его
> поддержку на Windows 7 (инсталлятор падает на `GetPackagesByPackageFamily`).
> Chrome app-mode — единый путь для всех ПК.

## Установка на операторский ПК
1. Убедись, что стоит **Google Chrome**. На **Win 7** — последняя рабочая версия
   **Chrome 109** (более новые не ставятся на Win 7).
2. Скопируй `ndpi-pult.bat` на ПК (напр. в `C:\ndpi\`).
3. Если адрес бокса не `192.168.0.127` — открой `.bat` в блокноте и поправь
   строку `set BOX=...`.
4. Двойной клик по `ndpi-pult.bat` → откроется окно пульта.

## Ярлык на рабочий стол
ПКМ по `ndpi-pult.bat` → «Отправить» → «Рабочий стол (создать ярлык)».
Можно сменить иконку в свойствах ярлыка.

## Автозапуск с Windows (по желанию)
`Win+R` → `shell:startup` → положи туда ярлык на `ndpi-pult.bat`.

## Заметки
- `--user-data-dir` — отдельный профиль Chrome, не мешает обычному браузеру
  оператора.
- Логин/JWT оператора хранит сам веб-пульт (как в обычном браузере).
- always-on-top у Chrome нет из коробки. Если нужно, чтобы окно всегда было
  поверх — добавим AutoHotkey-скрипт (скажи).
- Старый каталог `apps/operator-desktop` (Tauri) больше не используем.
